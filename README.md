# Portfolio | Amine Tahri

A modern, responsive portfolio website showcasing full-stack engineering projects, approach, and technical expertise.

## Overview

This portfolio demonstrates:
- **Full Stack Engineering** - Backend systems to frontend interfaces
- **Distributed Systems** - Resilient architectures and microservices
- **Data-Driven Solutions** - AI/ML and intelligent algorithms
- **Clean Code** - Well-documented, maintainable software

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org) - React with server-side rendering
- **Language**: [TypeScript](https://www.typescriptlang.org) - Type-safe JavaScript
- **Styling**: [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS framework
- **Animations**: [GSAP](https://greensock.com/gsap) - Professional animations
- **Package Manager**: [Bun](https://bun.sh) - Fast JavaScript runtime

## Getting Started

### Prerequisites
- Node.js 18+ or Bun 1.0+

### Installation

```bash
git clone https://github.com/tahri-md/tahri-md.github.io.git
cd tahri-md.github.io
bun install
```

### Development Server

```bash
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build for Production

```bash
bun run build
bun run start
```
